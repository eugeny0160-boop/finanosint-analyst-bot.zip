# finanosint-analyst-bot

Автоматический аналитик для Telegram-канала @finanosint.

## Возможности
- Сохраняет уникальные посты из канала (дедупликация по заголовку)
- Отправляет аналитические записки:
  - Ежедневно
  - Еженедельно
  - Ежемесячно
  - Каждые 6 месяцев
  - Ежегодно
- Работает бесплатно на Render + Supabase

## Настройка

1. Добавьте бота как администратора в @finanosint (с правом "Публиковать сообщения" и "Читать сообщения")
2. Создайте проект в https://supabase.com
3. Выполните `supabase_schema.sql` в SQL Editor
4. Задеплойте на Render
5. Настройте переменные окружения
6. Подключите cron-job.org

## Переменные окружения

- `BOT_TOKEN` — токен Telegram-бота
- `CHANNEL_ID` — "@finanosint" или numeric ID (например: -1001234567890)
- `SUPABASE_URL` — URL проекта Supabase
- `SUPABASE_KEY` — anon public key (из Project Settings → API)

## Cron-задания (cron-job.org)

| Путь        | Расписание                     |
|-------------|-------------------------------|
| /daily      | Каждый день в 08:00 UTC       |
| /weekly     | Понедельник 08:00 UTC         |
| /monthly    | 1-го числа каждого месяца     |
| /halfyear   | 1 января и 1 июля             |
| /yearly     | 1 января                      |

Метод: **POST**
